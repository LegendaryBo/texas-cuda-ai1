This package contains:

  1. EvDBConnector - class that can comunicate with database.
  2. EvPopulationDistributor - interface of population distributor classes.
  3. EvEqualPopulationDistributor - implementation of EvPopulationDistributor. Assuming there were n slaves instance of this class divides population into n equal subpopulations. Each subpopulation contain approximately population_size/n individuals.
  4. EvPopulationState - wrapper for the population. It contains population and information whether contained population is evaluated or not.
  5. EvMaster - master in the master-slave model.
  6. EvEvaluator - evaluates population (in distributed manner).
  7. EvClientID - contains client's id.
